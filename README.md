# PDV-Cupom-Fiscal-Manager
Desenvolver um programa em linguagem de programação que simule um sistema de comunicação com impressoras de cupom fiscal da Elgin, utilizando funções específicas da biblioteca da impressora, laços de repetição, estruturas condicionais, funções personalizadas. O foco do projeto é integrar lógica de programação com um cenário prático de automação comercial.
